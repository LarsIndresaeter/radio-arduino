radio-arduino
=============

software for Linux and the `rf-nano <https://ardunic.com/product/RF-Nano-Board-NRF24L01-Wireless-Module>`_.

Features
--------

* Command line build for avr and x86 using conan and cmake.
* Same binary for gateway and node.
* Binary message protocol for serial and radio communication.
* Command line tools for interacting with gateway and node, including mqtt client.
