# raduino-gateway

the application `raduino-gateway` is used for communication with the gateway device.

```console
raduino-gateway
       -V : Verbose on
       -v : Verbose off
       -B : Blink command
       -S : SHA1 command 
       -H : HOTP command
       -R : get random bytes command
       -C : print counter values
       -D : debug command
       -t : disable transport encryption
       -T : enable transport encryption
       -Z : set device name
       -z : get device name
       -a : get device version
       -x : get statistics
       -j : read vcc
       -s : sleep
       -p : ping command
       -E : set encryption key
       -K : set transport key on device (command must be encrypted)
       -b : use transport key
       -r : set transport encryption required (command must be encrypted)
       -n : reboot gateway as node
       -u : unencrypted session (command must be encrypted)
       -h : print this text
```
