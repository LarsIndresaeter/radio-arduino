# raduino-device-ws2812b

## help menu

```console
raduino-device-ws2812b
           -K <key> : encrypt command with transport key
       -N <address> : wakeup node address
                 -V : Verbose on
        -w <string> : WS2812B <string>
                 -h : print this text
```

## commands used

- [ws2812b](../commands/ws2812b.md)

## description 

This app let you write values to 8 ws2812b RGB leds.
