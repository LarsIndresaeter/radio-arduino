# raduino-mqtt-client

## description

This app reads the accelerometer from a lsm303d accelerometer.

## help menu

```console
raduino-device-lsm303d
           -K <key> : encrypt command with transport key
       -N <address> : wakeup node address
                 -V : Verbose on
                 -r : read lsm303d from gateway
                 -R : read lsm303d from node
                 -h : print this text
```
