# raduino-device-ina219

## help menu

```console
raduino-device-ina219
           -K <key> : encrypt command with transport key
       -N <address> : wakeup node address
                 -s : read ina219 on gateway
                 -s : read ina219 on node
                 -n : ina219 stats for <N> seconds on gateway
                 -h : print this text
```

## commands used

- [ina219](../commands/ina219.md)

## description 

This app reads voltage and current from a ina219 power monitor chips.
