# raduino-i2c-bridge

## help menu

```console
raduino-i2c-bridge
               -K <key> : encrypt command with transport key
           -N <address> : wakeup node address
           -a <address> : device address
          -c <register> : control register address
            -r <length> : read <length> bytes
            -s <string> : write <string>
             -w <value> : write <value>
                     -V : Verbose on
                     -h : print this text
```
