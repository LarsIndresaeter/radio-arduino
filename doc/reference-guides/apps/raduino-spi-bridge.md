# raduino-spi-bridge

## help menu

```console
raduino-spi-bridge
            -K <key> : encrypt command with transport key
        -N <address> : wakeup node address
       -o <register> : offset SPI register
         -r <length> : SPI read <length> bytes
          -w <value> : SPI write command
                  -V : Verbose on
                  -h : print this text
```
