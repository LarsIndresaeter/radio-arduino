# raduino-device-nrf24l01

## help menu

```console
raduino-device-nrf24l01
           -K <key> : encrypt command with transport key
       -N <address> : wakeup node address
                 -i : initialize nrf24l01 on gateway
                 -r : set device in serial bridge as reader
                 -s : set device in serial bridge as sender
                 -h : print this text
```

## commands used

- [nrf24l01_init](../commands/nrf24l01_init.md)
- [nrf24l01_read](../commands/nrf24l01_read.md)
- [nrf24l01_write](../commands/nrf24l01_write.md)

## description 

this app interact with the nrf24l01 chip on rf-nano.
