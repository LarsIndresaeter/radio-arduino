# raduino-device-ds28b20

## help menu

```console
raduino-device-ds28b20
           -K <key> : encrypt command with transport key
       -N <address> : wakeup node address
                 -x : ds18b20 temperature sensor from gateway
                 -X : ds18b20 temperature sensor from node
                 -h : print this text
```

## commands used

- [ds18b20](../commands/ds18b20.md)

## description 

This app reads the temperature from a ds18b20 one-wire temperature sensor
